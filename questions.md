# В чем разница между методами append() и extend() для списков? Приведите примеры.
* append() добавляет один объект целиком в конец списка.
* extend() принимает итерируемый объект (список, строку, кортеж) и добавляет каждый его элемент по отдельности в конец списка.
## Примеры
* list_a = [1, 2, 3]
* list_b = [4, 5]
* list_c = [1, 2, 3]

### append
list_a.append(list_b)
print(list_a)  # Вывод: [1, 2, 3, [4, 5]] (вложенный список)


### extend
list_c.extend(list_b)
print(list_c)  # Вывод: [1, 2, 3, 4, 5] (элементы распакованы)
# Как безопасно получить значение из словаря, если ключ может отсутствовать?
*Безопасно получить значение из словаря в Python, избежав ошибки, можно с помощью метода .get(key, default). Он возвращает значение ключа, если он есть, или заданное значение по умолчанию (по умолчанию None), если ключ отсутствует.
# Что такое списковые включения (list comprehensions) и в чем их преимущества?
* Списковое включение (list comprehension) — это лаконичный способ создания списков на основе других итерируемых объектов.
## преимущества
* Читаемость: Код короче и понятнее (для простых случаев) после того, как привыкнешь к синтаксису.
* Скорость: Работают быстрее, чем ручной цикл for с append(), так как оптимизированы интерпретатором.
